# Khiops storage driver shared components

This repository is dedicated to host components useful for many or all [Khiops](https://github.com/KhiopsML) drivers.

Since the storage drivers must conform to [a common interface](./integrationtests/plugin.hpp), there is a single integration test suite [here](./integrationtests/).
